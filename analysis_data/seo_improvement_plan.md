## 🎯 改善戦略

### 【最優先】フェーズ1: メタディスクリプションとタイトルの最適化

#### 対象ファイル
1. `taxonomy-grant_category.php` - カテゴリーアーカイブ
2. `taxonomy-grant_prefecture.php` - 都道府県アーカイブ
3. `taxonomy-grant_municipality.php` - 市町村アーカイブ
4. `taxonomy-grant_purpose.php` - 目的別アーカイブ
5. `taxonomy-grant_tag.php` - タグアーカイブ
6. `archive-grant.php` - 助成金アーカイブ

#### 具体的な変更内容
**削除する定型文:**
- 「→ 【毎日更新】...」
- 「助成金インサイトが、国・自治の公募情報を速報でお届けします」
- その他、サイト共通の宣伝文句

**置き換える内容:**
- ページ固有の価値を伝える説明文
- 具体的な助成金の内容、金額、対象者
- ユーザーが知りたい情報（申請方法、締切、要件など）

#### 期待される効果
- CTR向上: 10%以上の改善
- エンゲージメント率向上: 50%→60%以上
- 平均滞在時間の増加: 32秒→60秒以上

---

### 【優先】フェーズ2: PCサイトのパフォーマンス改善

#### 調査項目
1. **Core Web Vitals の確認**
   - LCP (Largest Contentful Paint): 2.5秒以内が目標
   - FID (First Input Delay): 100ms以内が目標
   - CLS (Cumulative Layout Shift): 0.1以内が目標

2. **PCとモバイルのコンテンツ同一性**
   - モバイルファーストインデックス対応の確認
   - PC版とモバイル版の構造化データの一致

3. **レンダリングブロック要素**
   - 不要なJavaScript/CSSの遅延読み込み
   - 画像の最適化（WebP、lazy loading）

4. **レスポンシブデザインの検証**
   - PC画面での表示崩れがないか
   - 大画面での最適化

#### 期待される効果
- PC掲載順位の改善: 24位→10位以内
- PC経由のクリック率向上

---

### 【定期】フェーズ3: 効果測定と継続的改善

#### 測定指標（KPI）
1. **Google Search Console**
   - CTR（クリック率）
   - 平均掲載順位（デバイス別）
   - 表示回数
   - クリック数

2. **Google Analytics**
   - エンゲージメント率
   - 平均エンゲージメント時間
   - 直帰率
   - ページ/セッション

#### 測定サイクル
- 毎週: Search Consoleデータの確認
- 毎月: Analytics詳細分析
- 四半期: 総合レビューと戦略調整
