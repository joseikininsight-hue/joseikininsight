## 🎯 原因分析

### 原因1: メタディスクリプションの問題 ★★★★★（最重要）
**発見された問題例**:
```
【現在の表示】
「【毎日更新】日本全国の補助金・助成金・給付金の最新情報一覧。
助成金インサイトが、国・自治の公募情報を速報でお届けします。
地域やカテゴリでの絞り込みも可能。」
```

**問題点**:
- ❌ ページ固有の情報が一切ない
- ❌ 「【毎日更新】」は検索結果で表示されるとクリック率を下げる
- ❌ サイト名やサービス説明が中心で、ユーザーが求める情報がない
- ❌ すべてのアーカイブページで同じ定型文を使用

**改善案**:
```
【改善後の例】
「6次産業化に関する助成金・補助金を47件掲載。2025年度の最新募集情報、
申請要件、対象事業、助成金額、締切日を詳しく解説。農産物加工や直売所
開設に最大500万円の支援が受けられる制度も紹介。」
```

### 原因2: PCサイトの技術的問題 ★★★★☆
**仮説**:
1. **モバイルファーストインデックス対応の不備**
   - PC版とモバイル版のコンテンツが異なる可能性
   - 構造化データがPC版で正しく実装されていない

2. **Core Web Vitalsの問題**
   - PCでのページ読み込み速度が遅い
   - レイアウトシフト（CLS）が発生している
   - 大きな画像やJavaScriptがレンダリングをブロック

3. **レスポンシブデザインの問題**
   - PC画面での表示崩れ
   - フォントサイズや余白の最適化不足

### 原因3: タイトルタグの最適化不足 ★★★☆☆
一部のページでタイトルが長すぎたり、キーワードが適切に配置されていない
